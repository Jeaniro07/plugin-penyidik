# Plugin Penyidik — Ditreskrimum Polda Sulawesi

Repositori ini berisi kustomisasi plugin **Asisten Penyidik Polri** yang dikonfigurasi untuk Ditreskrimum Polda Sulawesi.

## Instalasi

Download file `asisten-penyidik-polri.plugin` lalu drag & drop ke Claude Desktop (Cowork mode).

## Kustomisasi Satuan

- **Satuan**: Direktorat Reserse Kriminal Umum (Ditreskrimum)
- **Polda**: Sulawesi
- **Perkara Prioritas**: Pidana Umum, Tipidkor, Narkoba, Siber/ITE

Edit `asisten-penyidik-polri/config/profil-satuan.md` untuk memperbarui detail satuan.

## Fitur Plugin

- /pertanyaan-tersangka — Interogasi tersangka berdasarkan unsur pasal
- /pertanyaan-saksi — Pemeriksaan saksi
- /pertanyaan-pelapor — Pemeriksaan pelapor
- /pertanyaan-korban — Pemeriksaan korban
- /pertanyaan-ahli — Pemeriksaan ahli
- /analisis-bukti — Analisis barang bukti (GraphRAG + What-If)
- /review-kasus — Review berkas (multi-agent: Penyidik + Jaksa + Hakim)

## Dasar Hukum

- PERKABA No. 1 Tahun 2022
- KUHP Baru (UU No. 1/2023 jo. UU No. 1/2026)
- KUHAP Baru (UU No. 20/2025)
